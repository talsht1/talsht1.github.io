About Me

I'm a Senior Test Automation Engineer and SDET with a deep focus on building scalable, maintainable test automation systems that integrate tightly with CI/CD pipelines and engineering workflows. I've worked across frontend, backend, API, and distributed-system testing in complex, high-velocity environments.

I design and own automation infrastructure end-to-end — from framework architecture and test data management to Jenkins and GitHub Actions pipelines, Docker containers, and Kubernetes deployments.

I'm currently moving toward DevOps-adjacent roles, with hands-on training in infrastructure tooling (Terraform, Helm, Kubernetes) and a focus on release reliability, system testability, and engineering quality.
